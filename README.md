# Game-Classification

Implementing game classification based on unsupervised data

## Data:
* Extracted 200 games from snowflake DataBase

## Preprocessing:
* Lower casing
* Remove Stop words
* Remove Special characters
* Stemming
* Tokenizer

## Algorithms:
1. TF-IDF
2. Word2Vec
3. Bag of Words
4. Glove
5. Lbl2Vec
